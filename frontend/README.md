# 校园闲置物品智慧流转平台 前端
## 技术栈
React + Vite + TS + Ant Design + React Router + Zustand + Axios

## 环境要求（自行下载）
Node.js >= 18 LTS

## 运行命令
在终端执行以下命令
1. 安装依赖
npm install
2. 本地启动
npm run dev
启动成功后，可在浏览器访问：http://localhost:5173/
3. 打包部署
关闭当前终端，在新终端执行：
npm run build

## 平台功能
校园二手闲置发布、商品浏览、物品交换、求购、用户私信、个人中心

## 目录说明
src/utils axios请求封装
src/store 全局状态管理
